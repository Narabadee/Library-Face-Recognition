# 🚀 Face Scanner - Quick Start Guide

Welcome! This guide explains how to start the system with just one click, whether you are on Windows or Mac.

---

## 🪟 Windows (Current Setup)
1.  Open the project folder.
2.  Double-click on **`RUN_WINDOWS.bat`**.
    *   This will check your environment, install any missing pieces, and start the local server.
3.  The console will show your **Network URL** (e.g., `http://192.168.1.XX:5000`).

---

## 🍏 Mac (Migration)
1.  Copy the project folder to your Mac.
2.  Double-click on **`START_MAC.command`**.
    *   *Note: If macOS says the script can't be opened because it is from an unidentified developer, right-click the file and select 'Open'.*
3.  The system will automatically set itself up and show you the same **Network URL**.

---

## 📱 Mobile Access (Library Demo)
1.  Connect your Mac/Laptop to the Library Wi-Fi.
2.  Ensure your phone/tablet is on the **same Wi-Fi**.
3.  Open the browser on your phone and type the **Network URL** shown in the terminal.

---

## 🛠️ Troubleshooting
*   **Camera Permission**: Windows/macOS will ask for permission the first time you run the script. Click **Allow**.
*   **Python Handling**: You don't need to install Python manually anymore. `RUN_WINDOWS.bat` will detect it and install the correct version for you if it's missing or too old.
